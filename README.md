# CyberSecurityChatbot

# Description
This chatbot educates users about cybersecurity topics such as phishing, passwords, malware, safe browsing, and online safety.

# Features
- Personalized user interaction
- ASCII art introduction
- Audio greeting
- Cybersecurity awareness responses
- Help menu system
- Exit functionality

# How to Run
1. Open project in Visual Studio
2. Build solution
3. Run program (Start button)

# Topics Covered
- Password security
- Phishing attacks
- Malware and viruses
- Safe browsing
- Cybersecurity basics
- Two-factor authentication
